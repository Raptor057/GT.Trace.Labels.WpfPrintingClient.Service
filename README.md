## Objetivos

### - Objetivos generales del proyecto.
- Eliminar la recurrencia de etiquetas no impresas en las líneas de producción.
### - Objetivos específicos.
- Eliminar la recurrencia de etiquetas no impresas en las líneas de producción provocador por el programa ``Label Printing Client`` debido a que los operadores no lo abrían al iniciar el sistema / lo cerraba accidentalmente o abrían más de una vez, lo cual esto causaba que no se imprimieran las etiquetas o se imprimieran etiquetas de más.

## Instalación

Poner como un servicio de arranque de windows en las PC

## Uso

Se usa para evitar que los operadores no tengan que abrir el programa, y lo cierra en automático si lo abren más de una vez.
